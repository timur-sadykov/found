How it stays safe (without bureaucracy)

  • No addresses in public posts. Meet in a public place.
  • Proof is required: exact engraving/photo/receipt/detail.
  • Found posts should hide 1 key detail (so scammers fail).
  • This is a platform, not a police service.
Anti-scam rule (simple):
If you found an item, do NOT publish one unique detail. Ask the claimant to tell you that detail first.
