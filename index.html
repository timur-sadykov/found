Anti-scam rules

  • Never publish full unique details (serial number, full engraving, etc.).
  • Ask claimants to describe 2–3 hidden details (scratches, exact text, missing stone, etc.).
  • Meet in a public place (mall security / police station) for handover.
  • If in doubt — don’t hand it over.
